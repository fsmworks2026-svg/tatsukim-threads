プライバシーポリシー

本アプリ「tatsukim-threads」（以下「本アプリ」）は、Threads への自動投稿を目的とした個人利用のアプリです。

収集する情報

本アプリは、Threads API を通じて以下の情報にアクセスします。

  • Threads ユーザーID
  • Threads アクセストークン

これらの情報は、投稿の自動送信のみに使用します。

情報の利用目的

取得した情報は、本アプリの運営者（アカウント所有者）による Threads への自動投稿のためのみに使用します。第三者への提供、販売、共有は一切行いません。

情報の保管

アクセストークン等の認証情報は GitHub Secrets に暗号化して保管されます。ログファイルや外部サービスへの送信は行いません。

第三者サービス

本アプリは以下の外部サービスを利用します。

  • Meta Threads API：投稿の送信
  • Amazon アソシエイト：アフィリエイトリンクの生成
  • GitHub Actions：自動実行環境
